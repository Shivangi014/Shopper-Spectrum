# 🛒 Shopper Spectrum

Customer Segmentation and Product Recommendation System using Machine Learning.

---

## Features

- Interactive Dashboard
- Exploratory Data Analysis
- RFM Analysis
- KMeans Clustering
- Product Recommendation
- Streamlit Application

---

## Project Structure

```
Shopper-Spectrum/
│
├── app.py
├── train.py
├── utils.py
├── requirements.txt
├── README.md
├── online_retail.csv
│
├── models/
│   ├── scaler.pkl
│   ├── kmeans_model.pkl
│   ├── similarity.pkl
│   └── product_list.pkl
```

---

## Installation

```bash
pip install -r requirements.txt
```

---

## Train Models

```bash
python train.py
```

---

## Run Application

```bash
streamlit run app.py
```
